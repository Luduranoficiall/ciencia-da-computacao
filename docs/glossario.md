# Glossário PT ↔ EN

Termos que aparecem com frequência na **grade**, na **bibliografia** em inglês e em **entrevistas**. Não é definição formal de exame — é âncora rápida.

| Português (uso comum) | Inglês | Nota breve |
|------------------------|--------|------------|
| Algoritmo | Algorithm | Procedimento finito e bem definido para resolver um problema. |
| Complexidade (tempo/espaço) | Time/space complexity | Custo assintótico; muito associado a *Big O*. |
| Notação O grande | Big O notation | Limite assintótico superior do crescimento do custo. |
| Estrutura de dados | Data structure | Forma de organizar dados (vetor, fila, árvore, hash, …). |
| Tabela hash / dispersão | Hash table | Mapeamento chave → valor com função hash. |
| Fila / pilha | Queue / stack | FIFO vs LIFO. |
| Grafo | Graph | Vértices e arestas; direcionado ou não. |
| Compilador | Compiler | Traduz código-fonte para outra forma (máquina, bytecode, …). |
| Lexer / analisador léxico | Lexer / scanner | Quebra o texto em *tokens*. |
| Parser / analisador sintático | Parser | Verifica estrutura segundo uma gramática. |
| Autômato finito | Finite automaton | Modelo de memória finita para linguagens regulares. |
| Máquina de Turing | Turing machine | Modelo teórico de computação universal. |
| Processo | Process | Programa em execução com espaço de endereçamento próprio (SO). |
| Fio de execução / thread | Thread | Fluxo de execução dentro de um processo. |
| Deadlock | Deadlock | Espera circular por recursos; ninguém progride. |
| Chamada de sistema | System call | Interface pedida ao núcleo (kernel), ex.: ler ficheiro. |
| Latência | Latency | Tempo até a primeira resposta útil. |
| Débito / vazão | Throughput | Trabalho concluído por unidade de tempo. |
| Idempotência | Idempotency | Repetir a mesma operação não altera o estado além do primeiro efeito. |
| *Backoff* exponencial | Exponential backoff | Aumentar espera entre retries para não agravar sobrecarga. |
| API | API | Contrato exposto por um serviço (HTTP, biblioteca, SO). |
| CI / CD | CI / CD | Integração e entrega/implantação contínuas. |
| TDD | TDD | Test-driven development: teste antes ou em ciclo curto com o código. |
| *Threat modeling* | Threat modeling | Identificar ameaças e superfícies de ataque de um sistema. |
| Contentor | Container | Empacotamento isolado de aplicação (ex.: Docker/OCI). |
| Observabilidade | Observability | Entender estado interno via logs, métricas, *traces*. |
| SLI / SLO | SLI / SLO | Indicador e objetivo de nível de serviço. |
| Normalização | Normalization | Reduzir redundância e anomalias em esquema relacional. |
| Índice (BD) | Index (DB) | Estrutura para acelerar procura; custa espaço e escrita. |
| Gradiente | Gradient | Vetor de derivadas; base de otimização em ML. |
| *Overfitting* | Overfitting | Modelo que decora o treino e generaliza mal. |
