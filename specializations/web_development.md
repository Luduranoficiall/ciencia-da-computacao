# Especialização: Desenvolvimento web (full stack sério)

## Pré-requisitos mínimos

- Programação II, OO, banco de dados, redes, engenharia de software.

## Trilha sugerida

1. **Web fundamentals** — HTTP, cookies, CORS, caching, segurança (OWASP para web).
2. **Front** — HTML semântico, CSS, acessibilidade, um framework moderno (React, Vue, Svelte…).
3. **Back** — API REST ou GraphQL; autenticação (sessions vs JWT); validação de entrada.
4. **Persistência e migrações** — modelo relacional + índices; transações.
5. **Deploy** — reverse proxy, HTTPS, variáveis de ambiente, logs.

## Recursos

- [MDN](https://developer.mozilla.org/), [freeCodeCamp](https://www.freecodecamp.org/learn).
- OWASP [ASVS](https://owasp.org/www-project-application-security-verification-standard/) como checklist.

## Capstone

- **A:** App completa com auth, papéis, testes E2E críticos e CI.
- **B:** BFF ou API pública versionada com OpenAPI e rate limit documentado.

## Prova de nível

Repositório com ADR curto (1 página) sobre escolha de stack; threat model de uma rota sensível; Lighthouse ou axe em fluxo principal.
